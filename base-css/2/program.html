Как работать на курсе?

  1. Изучить теги
  2. Попрактиковаться
  3. Выполнить домашнее задание

Описание курса

В первой части вы познакомитесь со стандартами Web 2.0 научитесь работать с маетами и формировать код, освоите адаптивную верстку, а по итогу — сможете создавать быстрые и удобные сайты, которые точно понравятся пользователям.

Вы поймете назначение основных тегов HTML, сможете без усилий стилизовать и размещать элементы с помощью CSS и сверстаете свой первый макет.

Вторая часть курса сделает из вас настоящего мастера. Вы изучите сборщики проектов, научитесь верстать email-рассылки, разберетесь с анимацией на сайте на сайте.

Помимо этого, вы сверстаете фрагмент из интернет-магазина - главную страницу каталога и страницу товара!
